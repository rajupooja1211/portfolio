MOBILE APP PROJECT

TIDY TOWN

Tidy Town is a playful, cartoon-style mobile app designed to teach kids the importance of proper waste sorting through interactive learning and gamified challenges.
The app combines animations, sound, and reward-based progression to create a fun, engaging experience where learning feels like play.

Year 2025
Industry EdTech
Status Completed

Tech Stack

Flutter, Dart, Lottie, SharedPreferences, Firebase (Gamified waste-sorting app with learning modules and cartoon-style UI)

THE PROBLEM

Many recycling and sustainability apps are too text-heavy or only available in English making them inaccessible for younger children and non-English-speaking families. There’s a lack of bilingual, child-friendly apps that combine environmental learning with visuals, interactivity, and progress-based motivation. Kids learn best through play but there were few mobile-first tools that deliver that in both English and Spanish.

THE SOLUTION

Tidy Town reimagines environmental education as a fun, bilingual journey. The app blends visual learning, gamification, and interactive UI elements to teach kids how to recycle, compost, and sort waste in English and Español.

Key features include:

  • Bilingual Learning – Every module is available in both English and Spanish, with plans for voice narration

  • Unlockable Game Flow – Kids complete learning activities before playing, reinforcing understanding first.

  • Progress Saving – User progress is stored locally using SharedPreferences, so they can pick up where they left off.

  • Playful UI – Cartoon-style layout, animated icons, soft gradients, and rounded navigation for a joyful feel.

  • Sound + Language Toggle – Easy controls for switching language and turning sound on/off.

  • Future: Location-Based Themes – Dynamic backgrounds that match user location (e.g., city, forest, beach).

  • Reward Ideas (Planned) – Badges and feedback for completing tasks and answering quizzes correctly.

    • The app is built using Flutter to support both iOS and Android devices with a single codebase.

    THE OUTCOME

    Tidy Town is well underway, with core learning modules, bilingual toggle, persistent storage, and game unlock logic in place Tidy Town. The app is already being tested with bilingual UI text and playful learning flow laying the foundation for a more inclusive, engaging learning experience for kids from multilingual households.
